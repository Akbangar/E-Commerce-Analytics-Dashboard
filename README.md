# 📊 E-Commerce Analytics Dashboard

## 📌 Project Overview
This project presents an end-to-end E-Commerce Analytics solution designed to analyze business performance using real-world transactional data.  
It combines SQL for data extraction, Excel for preprocessing, and Power BI & Tableau for interactive dashboards and visualization.

---

## 🎯 Objectives
- Analyze sales and profit performance  
- Understand customer behavior and segmentation  
- Evaluate delivery performance and its impact on customer reviews  
- Identify top-performing products and categories  
- Enable data-driven decision-making  

---

## 🛠 Tools & Technologies
- **SQL** – Data extraction and transformation  
- **Power BI** – Dashboard creation and KPI tracking  
- **Tableau** – Advanced data visualization  
- **Excel** – Data cleaning and preprocessing  

---

## 📊 Key Features
- KPI Cards: Total Sales, Profit, Customers, Sellers  
- Sales Analysis (Weekday vs Weekend performance)  
- Payment Method Analysis  
- Customer Segmentation  
- Delivery Performance Analysis  
- Product Category Insights (Top Products)  
- Interactive filters and dynamic dashboards  

---

## 📈 Key Insights
- Weekday sales generate higher revenue compared to weekends  
- Credit cards are the most preferred payment method  
- Faster delivery improves customer review scores  
- A few product categories contribute to the majority of sales  
- Customer location influences purchasing behavior
